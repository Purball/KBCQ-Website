🌐 How to Publish Kim’s Blue Crab Quest Online

This beginner-friendly guide shows you how to publish your site using Firebase or Netlify.

1️⃣ Preview Your Site Locally

  1. Open your folder: C:\Users\CAPTA\OneDrive\Desktop\crab\KBCQ-Website
  2. Double-click index.html to open in your browser.
  3. For live updates, install VS Code Live Server.

2️⃣ Deploy with Firebase Hosting

📖 Firebase Hosting Documentation

  1. Go to Firebase Console and click Add Project.
  2. Enable Hosting and complete setup.
  3. Run the following commands in your terminal:
npm install -g firebase-tools
firebase login
firebase init
firebase deploy

Your site will be live at https://kbcq.web.app.

3️⃣ Deploy with Netlify

  1. Go to Netlify.com and sign in.
  2. Drag and drop your KBCQ-Website folder into the upload area.
  3. Your site will be instantly published.
